- 맨 뒤에 삽입시는 N+1 단계

## 2.2 정렬된 배열의 검색

- 선형 검색 : 왼쪽에서 오른쪽으로 한 번에 한 셀씩 확인하는 방법

- 정렬된 배열의 선형 검색

```js
function linearSearch(array,searchValue){
    array.forEach((element,index)=>{
        if(element === searchValue) return index
        if(element > searchValue) break
    })
    return null
}
```
