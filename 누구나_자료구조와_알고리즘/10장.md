# 10장 재귀를 사용한 재귀적 반복

## 10.1 루프 대신 재귀

- 루프를 사용할 수 있는 경우라면 대부분 재귀도 가능하다.

- 아래는 임의의 숫자를 받아서 0까지 카운드다운하는 함수이다.

```js
function countDown(number) {
  console.log(number);
  if (number === 0) return;
  countDown(number - 1);
}
```

## 10.2 기저 조건

- 기저 조건 : 함수가 반복되지 않는 경우

- 위의 예제에서는 0이 기저 조건이다.

- 무한대로 호출되지 말아야 할 조건
