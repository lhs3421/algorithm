# 12장 동적 프로그래밍

## 12.1 불필요한 재귀 호출

- 아래의 함수는 최대값을 찾는 재귀 함수이다.

- 하지만 불필요한 재귀 호출이 일어난다.

```js
function max(arr) {
  if (arr.length === 1) return arr[0];

  if (arr[0] > max(arr.slice(1))) {
    return arr[0];
  } else {
    return max(arr.slice(1));
  }
}
```

## 12.4 하위 문제 중첩

- 아래 함수는 피보나치 함수이다.

- 하지만 자기 자신을 2번 호출하므로 비효율적이다.

```js
function fib(n) {
  if (n === 0 || n === 1) return n;

  return fib(n - 2) + fib(n - 1);
}
```
