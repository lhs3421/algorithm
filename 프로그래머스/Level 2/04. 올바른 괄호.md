# 올바른 괄호

## 문제 링크

- https://school.programmers.co.kr/learn/courses/30/lessons/12909

## 나의 문제 풀이

```js
function solution(s) {
  const split = s.split("()");
  const filter = split.filter((a) => a !== "");
  for (let i = 0; i < filter.length; i++) {
    if (filter[i] === "(" && filter[i + 1] === ")") return true;
    else return false;
  }
  if (filter.length === 0) return true;
}
```
