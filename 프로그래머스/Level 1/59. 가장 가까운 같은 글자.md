# 가장 가까운 같은 글자

## 문제 링크

- https://school.programmers.co.kr/learn/courses/30/lessons/142086

## 나의 문제 풀이

```js
function solution(s) {
  const answer = [];
  for (let i = 0; i < s.length; i++) {
    if (s.indexOf(s[i]) === i) {
      answer.push(-1);
    } else {
      const index = s.slice(0, i).lastIndexOf(s[i]);
      answer.push(i - index);
    }
  }
  return answer;
}
```
