## 나의 문제 풀이

```js
function solution(number) {
  for (let i = 0; i < number.length - 2; i++) {
    for (let j = i + 1; j < number.length - 1; j++) {
      for (let k = j + 1; k < number.length; k++) {}
    }
  }
}
```
